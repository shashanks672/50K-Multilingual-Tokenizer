# 🤖 50K Multilingual BPE Tokenizer

A multilingual BPE tokenizer built using English, Hindi, Kannada, and Tamil Wikipedia corpora.

## Features

- 50,000 vocabulary size
- Supports:
  - English
  - Hindi
  - Kannada
  - Tamil
- Interactive Streamlit web interface
- Displays generated tokens and token IDs

## Tech Stack

- Python
- Hugging Face Tokenizers
- Streamlit

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```
